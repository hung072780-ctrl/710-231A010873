Giới Thiệu

Trang web này là sản phẩm bài tập thực hành môn Phát triển Ứng dụng Web, thuộc lớp 710 (Mã số Sinh viên: 231A010873). Mục tiêu của các bài tập là rèn luyện kỹ năng sử dụng các công nghệ cốt lõi của Web: **HTML, CSS (với Tailwind CSS cho tiện ích), và JavaScript**.

Các Kỹ Năng Ứng Dụng

  • **HTML5:** Cấu trúc và ngữ nghĩa trang web.

  • **JavaScript ES6+:** Xử lý DOM, logic bất đồng bộ (implicit Promise trong `setInterval`), quản lý state, và lưu trữ dữ liệu cục bộ (`LocalStorage`).
  • **Kiến trúc:** Phân tách mã (Code Separation) giữa HTML, CSS (`style.css`), và JS (`script.js`).
